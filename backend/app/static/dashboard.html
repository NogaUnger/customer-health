Customer Health

Scores blend Login Frequency, Feature Adoption, Ticket Volume, Invoice Timeliness, and API Usage Trend. Higher is better (0–100).

What makes up the score?
  • Login Frequency: share of active days (size/segment-normalized).
  • Feature Adoption: # of distinct key features used in last 30 days.
  • Ticket Volume: fewer tickets in last 30 days → higher score (normalized).
  • Invoice Timeliness: on-time vs late invoices over the last .
  • API Usage Trend: last 7 days vs the 7 days before (up=better).
